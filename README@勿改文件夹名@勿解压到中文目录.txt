==WBBamp系统提示==
1.请不要将WBBamp解压到中文目录，避免Apache出错；
2.如需修改WBBamp文件夹本名，需要相应修改部分内部配置；
3.start.bat全自动启动Apache+Mysql，另外两个文件同理；
4.实测运行占用总内存<20MB，httpd进程*2，mysqld进程*1；
5.PhpMyAdmin高级功能已开启；
6.系统可以放在U盘中运行，不留服务；
7.默认网站目录为/www/

==WBBamp各组件版本==
Php：5.4.17，配置文件位置：/php/php.ini
Apache：2.4.6，配置文件位置：/apache2/conf/httpd.conf
MySQL 5.5.39，配置文件位置：/mysql/my.ini
PhpMyAdmin 4.0.4.2
